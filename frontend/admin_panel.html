埋点说明

用户日活：统计当日至少发起过一次需登录接口（携带用户 JWT）的去重账号数，不含纯浏览静态页、不含管理后台。接口调用：按自然日聚合各 API 路径与业务分类（税务记录、消息中心、认证注册等）；含管理后台与公开接口。登录：记录账号密码登录成功/失败（不含注册）；单条流水、分页查询见侧栏登录流水。设备：客户端每次请求通过请求头 X-Client-Device 上报 JSON（含 client_id、机型字段可由 App WebView 注入）；服务端按 client_id 聚合同一终端，并在每次已登录接口调用时刷新详情与「接口同步次数」。
