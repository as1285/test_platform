下载描述文件

安装步骤（iPhone / iPad）

  1. 用 Safari 打开本页，点击下载描述文件；出现提示时选择允许。
  2. 打开 设置 → 通用 → VPN与设备管理（部分系统为「描述文件与设备管理」），找到刚下载的描述文件并安装。
  3. 安装成功后，返回桌面即可进入 APP。
  4. 具体操作请向下查看操作视频。
